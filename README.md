# Ashitosh Ghodake Portfolio

Modern, responsive React + Tailwind portfolio website for a Cyber Security Analyst / SOC Analyst.

## Stack

- React
- Vite
- Tailwind CSS
- Lucide React icons

## Run locally

```bash
npm install
npm run dev
```

## Build for deployment

```bash
npm run build
```

## Personal details to update

- Email: `src/data/portfolio.js`
- College name: `src/data/portfolio.js`
- Education duration: `src/data/portfolio.js`
